相关命令行解释

基本的命令行
| 命令 | 解释 |
| ——– | :—–: |
| git status|查看已经被修改，但是没有被提交的文件|
| git diff |查看具体的修改记录（diff = different）|
| git log |显示从最近到最远的提交日志（可以看得见 SHA 码；如果输出信息太多可以加上 –pretty=oneline|
| git checkout – file |可以放弃工作区的修改（–很重要，没有–，就变成了“切换到另一个分支”的命令）|
| git push -u origin master|将当前分支 master 推送到远程|
| git remote add origin git@server-name:path/repo-name.git|关联一个远程库|
